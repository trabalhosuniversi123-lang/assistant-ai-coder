# 🤖 Assistant AI Coder

O **Assistant AI Coder** é um assistente virtual desenvolvido em **Python** utilizando **Streamlit** e a **Groq API** para responder dúvidas sobre programação Python de forma clara, estruturada e didática.

## 🚀 Funcionalidades

- Explicação conceitual sobre temas da linguagem Python.
- Geração de exemplos de código comentados.
- Explicação detalhada do funcionamento do código.
- Indicação da documentação oficial para aprofundamento.
- Interface web simples e intuitiva.

## 🛠️ Tecnologias Utilizadas

- Python
- Streamlit
- Groq API
- Engenharia de Prompt (Prompt Engineering)

## ▶️ Como executar o projeto

### 1. Clone o repositório


### 2. Acesse a pasta

```bash
cd assistant-ai-coder
```

### 3. Instale as dependências

```bash
pip install -r requirements.txt
```

### 4. Execute a aplicação

```bash
streamlit run app.py
```

## 🔑 Configuração

Para utilizar a aplicação, é necessário possuir uma chave de API da **Groq**.

Ao iniciar o sistema, informe sua chave no campo disponível na barra lateral.
